Scoring Matrix

Choose a predefined scoring matrix. A scoring matrix is a way of scoring particular alignments of bases or amino acids rather than using a single score for any match or mismatch. It is a way to set specific score when, for example, the amino acid A matches A. See the help menu for a more detailed explanation.

Standard DNA Scoring Matrix:

A 1 -1 -1 -1
T -1 1 -1 -1
G -1 -1 1 -1
C -1 -1 -1 1
  A T G C

This specific matrix gives us the standard 1 score for match, and -1 for mismatch.
