Sequences

Deoxyribonucleic Acid (DNA)

DNA is a doubled stranded molecule that holds the genetic instructions used in the development, functionality, reproduction, and growth of an organism. For the purposes of sequencing, DNA is shown in a single sequence of the bases Guanine (G), Cytosine (C), Adenine (A), and Thymine (T).

DNA Example: GCTACTAGGTCA

Bases:

Ribonucleic Acid (RNA)

RNA is a single-stranded copy of a section of DNA, much like a recipe from a cookbook. Like DNA, it is essential for life. RNA uses the same bases, but Thymine (T) is replaced with Uracil (U). This is because Thymine is very stable, but Uracil is cheaper; DNA needs to be stable, whereas RNA is a short-lived copy, and thus does not need the extra stability. In some systems, RNA sequences are written with T instead of U because of the irrelevance in sequencing.

Example:

Sequence Type Sequence
DNA GCTACTAGGTCA
RNA GCUACUAGGUCA

Protein

Proteins are large biomolecules with complex structures that are constructed using RNA. Despite the complexity, a protein is just a sequence of amino acids. Its structure and function are determinedby this sequence. Each amino acid is made from a one or more different codons. A codon is simply just a sequence of three bases. Each amino acid may be the result one, none, or many codons. Codons that don't correspond to an amino acid are called stop codons, which terminate the sequence.

Example:

Sequence Type Sequence
DNA GCTACTAGGTCA
RNA GCUACUAGGUCA
Protein ATRS

Sequence Alignment Algorithms

Sequence alignment is the process of aligning two or more sequences to be able to compare them. It can give us important information such as how similar different organisms are. Alignment is done by inserting gaps into the sequence, thus allowing bases of one sequence match the base of the other sequence. What score to assign to matches, mismatches, and gaps is chosen experimentally. For algorithm explanation we will set the score of a match to 1 for Needleman-Wunsch, 2 for Smith-Waterman, and the score of a mismatch or gap as -1.

Needleman-Wunsch

Developed by by Saul B. Needleman and Christian D. Wunsch, this algorithm was designed to compared biological sequences. It outputs the global alignment, which means it tries to align every base.

In aligning sequences GCATGCT and GATTACA, we start off with this grid:

To compute the value for this cell, we consider all 3 options: taking from the top or left cell, marked in red, would introduce a gap in one of the sequences, and so would give a score of -1-1=-2, since we award a score of -1 for a gap. If we take from the diagonal, marked in blue, we have to see whether the two bases match. In this case they do, so we would give a score of 0+1=1.

So we have either -1, -1, or 1 to choose from. Always take the highest scoring possibility, so the cell would be a value of 1. If there were two routes that gave the same, but highest score, then this may produce two different best alignments. Continue on until all cells are filled.

Scoring

Scoring Matrix

When using a single score for a match or a mismatch, it assumes that all matches and mismatches yield the same score. This is not always true, especially for proteins. That is, the amino acid A matching A may not yield the same score as V matching V, similarly with mismatches. One way to show this is to use a scoring matrix.

Standard DNA Scoring Matrix:

A 1 -1 -1 -1
T -1 1 -1 -1
G -1 -1 1 -1
C -1 -1 -1 1
  A T G C

This specific matrix gives us the standard 1 score for match, and -1 for mismatch.

Gap Penalties

In application, when applying a relatively big initial gap penalty that decreases for multiple gaps in a row provides better results in bioinformatics. That is, if one aligned sequence is, say, A-C-G, it is better to have it penalized more than say, A--CG.

Outputs

Score

The score of the alignment.

Identity Value (id-value)

The extent to which two sequences have the same residues at the same positions in an alignment, often expressed as a percentage.
